# Simple Python Calculator

## Description
This project is a simple command-line calculator written in Python.  
It allows users to perform basic arithmetic operations such as:

- Addition
- Subtraction
- Multiplication
- Division

The program prompts the user to enter two numbers and an operator, then calculates and displays the result.

## Requirements
- Python 3.x installed
